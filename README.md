# SkillSystemProject - Hệ Thống Cây Kỹ Năng Tùy Biến (Modular Skill Tree)

Dự án game Unity học tập, tập trung vào việc phát triển hệ thống kỹ năng modular (Form + Element + Modifier) cho phép người chơi tự tạo và kết hợp siêu năng lực (Emergent Gameplay).

## 🚀 Tính Năng Chính
* **Hệ thống Skill Tree tĩnh:** Dữ liệu kỹ năng quản lý bằng `ScriptableObject`.
* **Cơ chế Modular:** Tách biệt Hình thái (Form), Nguyên tố (Element), và Bộ biến đổi (Modifier).
* **Emergent Interactions:** Các nguyên tố và thuộc tính vật lý phản ứng chéo sinh ra hiệu ứng mới.

## 📁 Cấu Trúc Dự Án
* [Game_Knowledge_Base/](Game_Knowledge_Base/): Nơi lưu trữ tài liệu tri thức game, thiết kế mechanic và sprint task.
* [AGENT/](AGENT/): Chứa các chỉ dẫn và quy tắc cấu hình cho AI Assistant (Antigravity).
* `Assets/`: Chứa mã nguồn game và các Asset Unity.

## 🛠️ Công Nghệ Sử Dụng
* Engine: Unity
* Ngôn ngữ: C#
